## Контекст
Модуль "Frontend Генератор Пейлоадів" забезпечує користувацький інтерфейс для створення та налаштування різних типів пейлоадів. Він дозволяє користувачам вибирати параметри, переглядати шаблони та генерувати пейлоади для різних платформ.

## Структура
Основні файли модуля:
- `CYBER_DASHBOARD_FRONTEND/js/payload_generator_ui.js`: Основний JavaScript файл для UI генератора пейлоадів.
    - `initializePayloadGenerator()`: Ініціалізація генератора пейлоадів.
    - `loadPayloadTemplates()`: Завантаження доступних шаблонів.
    - `handlePayloadGeneration()`: Обробка генерації пейлоаду.
    - `displayGeneratedPayload()`: Відображення згенерованого пейлоаду.
- `CYBER_DASHBOARD_FRONTEND/css/payload_generator.css`: Стилі для інтерфейсу генератора.
- `CYBER_DASHBOARD_FRONTEND/payload_generator.html`: HTML структура для розділу генератора.

## Поведінка
1. **Управління Шаблонами**:
   - Відображення списку доступних шаблонів.
   - Перегляд деталей шаблонів.
   - Фільтрація шаблонів за типом та платформою.
2. **Генерація Пейлоадів**:
   - Вибір параметрів для генерації.
   - Валідація вхідних даних.
   - Відображення прогресу генерації.
3. **Взаємодія з Backend**:
   - Відправка запитів на генерацію пейлоадів.
   - Отримання та відображення результатів.
   - Обробка помилок та сповіщень.
4. **UI Компоненти**:
   - Форми для вибору параметрів.
   - Елементи для відображення згенерованих пейлоадів.
   - Інструменти для копіювання та експорту.
